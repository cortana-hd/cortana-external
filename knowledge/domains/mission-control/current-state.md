# Mission Control Current State

Mission Control currently serves as the local operational dashboard for agents, runs, jobs, docs, approvals, and related control-plane telemetry.

Current shape:

- Next.js app under `apps/mission-control`
- raw source docs split between repo-level architecture docs and app-local notes
- canonical knowledge pages live under `knowledge/domains/mission-control/`
- Trading Ops now includes a live Polymarket tab with account state, pinned markets, top events, top sports, linked watchlists, and results
